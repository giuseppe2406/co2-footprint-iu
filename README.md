# CO₂-footprint-IU

Diese Webanwendung stellt fiktive CO₂-Emissionsdaten von Unternehmen und Ländern
in tabellarischer Form dar. Sie wurde im Rahmen einer Fallstudie umgesetzt und
dient der Demonstration grundlegender Funktionen einer interaktiven Website.

## Funktionen

- Responsive Darstellung für Desktop, Tablet und Smartphone
- Tabelle mit Filterfunktion nach Unternehmen und Ländern
- Sortierung nach Text- und Zahlenwerten
- Umschaltung der Leserichtung (LTR / RTL)
- Einfaches Impressum und Datenschutzerklärung für Studienzwecke

## Technologien

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript (Vanilla JS)

## Projektstruktur

- `index.html` – Hauptseite der Webanwendung
- `style.css` – Gestaltung und Layout
- `scripts.js` – Filter-, Sortier- und Interaktionslogik
- `impressum.html` – Impressum (Studienprojekt)
- `datenschutz.html` – Datenschutzerklärung (Studienprojekt)
- `logo.png`, `banner.png` – Grafische Elemente

## Nutzung

1. Repository klonen oder herunterladen  
2. Datei `index.html` im Browser öffnen  
3. Es ist keine zusätzliche Installation erforderlich  
  

## Hinweise

Die dargestellten CO₂-Daten sind frei erfunden und dienen ausschließlich der
Demonstration. Die Inhalte von Impressum und Datenschutzerklärung sind nicht
rechtlich verbindlich.
